ServiceStack Templates is an exciting new templating language that can be used as a fast, clean, portable alternative to Razor and Razor Pages that's better suited for maintaining content-heavy sever-generated websites, easily facilitates a live programming environment for .NET Apps, makes it easy to introspect the state of running .NET Apps, provides non-technical users a fun and intuitive way to generate text documents like email templates, and more!
